# Português Brasileiro — Regras Completas

## Proparoxítonas (TODAS acentuadas — sem exceção)

Toda palavra com sílaba tônica na antepenúltima posição recebe acento. Lista de palavras frequentemente esquecidas:

análise, específico, diagnóstico, estratégia, lógica, único, página, número, código, método, histórico, prático, índice, tópico, próximo, último, máximo, mínimo, ótimo, técnico, músico, público, república, acadêmico, econômico, simpático, fantástico, dinâmico, mecânico, eletrônico, filosófico, gráfico, automático, político, básico, clássico, trágico, lógico, prática, teórico, hipótese, parêntese, síntese, análogo, anônimo, autônomo, sinônimo, pêssego, lâmpada, sílaba, líquido, rápido, tímido, sólido, válido, módulo, cômodo, pálido, ângulo, cálculo, círculo, obstáculo, espetáculo, veículo, ridículo, artístico, doméstico, romântico, energético, genético, patético, estético, atlético, frenético, sintético, periódico, esporádico, aromático

### Mapa de correção (sem acento → com acento)

```
analise → análise (substantivo; "analise" como subjuntivo é sem acento)
especifico → específico (adjetivo; "especifico" como verbo é sem acento)
diagnostico → diagnóstico (substantivo; "diagnostico" como verbo é sem acento)
estrategia → estratégia
logica → lógica
unico → único
pagina → página
numero → número (substantivo; "numero" como verbo é sem acento)
codigo → código
metodo → método
historico → histórico
pratico → prático (adjetivo; "pratico" como verbo é sem acento)
publico → público (adjetivo/substantivo; "publico" como verbo é sem acento)
```

**ATENÇÃO**: Palavras como "analise", "diagnostico", "numero", "publico", "pratico", "especifico" podem ser formas verbais (sem acento) ou substantivos/adjetivos (com acento). SEMPRE verificar o contexto da frase antes de corrigir.

**Regra**: ç só aparece antes de A, O, U. Antes de E, I usa-se apenas C (sem cedilha): certo, cidade, cena, cigarro.

## Crase (à = a + a)

### USAR crase:
- Antes de palavras femininas com sentido de direção/destino: "Ir à escola", "Chegar à cidade"
- Em expressões fixas: "às vezes", "à noite", "à tarde", "à esquerda", "à direita", "à vontade", "à medida que", "à proporção que"
- Antes de "aquele/aquela/aquilo": "Refiro-me àquele assunto"
- "Referente à questão", "Graças à ajuda"

### NUNCA usar crase:
- Antes de verbo: "Começou a fazer", "Passou a entender"
- Antes de palavra masculina: "Fui a pé", "Andou a cavalo"
- Antes de pronomes pessoais: "Disse a ela"
- Antes de "uma": "Fui a uma loja"
- Entre palavras repetidas: "Cara a cara", "Frente a frente"

## Concordância verbal

| ERRADO | CORRETO |
|--------|---------|
| a gente vamos | a gente vai |
| faz dois anos (tempo) | faz dois anos (correto — verbo impessoal) |
| fazem dois anos | faz dois anos |
| houveram problemas | houve problemas |
| existem a possibilidade | existe a possibilidade |
